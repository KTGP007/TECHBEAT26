## 2. ✅ शुरू करने से पहले क्या चाहिए

शुरू करने से पहले, सुनिश्चित करें कि आपके पास ये हैं:

| आवश्यकता | कैसे जांचें |
|-----------|-------------|
| **Python 3.8+** इंस्टॉल | टर्मिनल खोलें → `python --version` टाइप करें |
| **VS Code** इंस्टॉल | Start Menu में "Visual Studio Code" खोजें |
| **स्क्रिप्ट फ़ाइल** `v2.py` | आयोजकों द्वारा दी जाएगी |

### Python इंस्टॉल करना (अगर नहीं है)

1. [python.org/downloads](https://www.python.org/downloads/) पर जाएं
2. Windows के लिए नवीनतम Python 3 डाउनलोड करें
3. **⚠️ ज़रूरी:** इंस्टॉलेशन के दौरान ✅ **"Add Python to PATH"** पर टिक करें
4. "Install Now" पर क्लिक करें
5. सत्यापित करें — Command Prompt खोलें और टाइप करें:
   ```
   python --version
   ```
   आपको कुछ ऐसा दिखेगा: `Python 3.12.x`

### VS Code इंस्टॉल करना (अगर नहीं है)

1. [code.visualstudio.com](https://code.visualstudio.com/) पर जाएं
2. Windows इंस्टॉलर डाउनलोड करें
3. डिफ़ॉल्ट सेटिंग्स के साथ इंस्टॉल करें
4. **सुझाव:** **Python extension** इंस्टॉल करें (अगला खंड देखें)

## 3. 🛠️ VS Code सेटअप

### चरण 1: Python Extension इंस्टॉल करें

1. VS Code खोलें
2. बाईं साइडबार में **Extensions** आइकन पर क्लिक करें (4 चौकोर जैसा दिखता है)
   - या `Ctrl + Shift + X` दबाएं
3. **"Python"** खोजें
4. **Microsoft** द्वारा बनाए गए पर **Install** पर क्लिक करें (पहला रिजल्ट होगा)

### चरण 2: Python Interpreter चुनें

1. `Ctrl + Shift + P` दबाएं (Command Palette खुलेगा)
2. **"Python: Select Interpreter"** टाइप करें
3. अपना इंस्टॉल किया हुआ Python वर्शन चुनें (जैसे `Python 3.12.x`)

## 4. 📂 प्रोजेक्ट फ़ोल्डर खोलना

### तरीका 1: VS Code से

1. VS Code खोलें
2. **File → Open Folder** पर जाएं
3. उस फ़ोल्डर में जाएं जहां `v2.py` सेव है
4. **Select Folder** पर क्लिक करें

### तरीका 2: File Explorer से

1. File Explorer में `v2.py` वाले फ़ोल्डर में जाएं
2. फ़ोल्डर में राइट-क्लिक करें
3. **"Open with Code"** चुनें (अगर उपलब्ध हो)

### तरीका 3: टर्मिनल से

```powershell
cd C:\path\to\your\folder
code .
```

> [!IMPORTANT]
> **हमेशा फ़ोल्डर खोलें, सिर्फ फ़ाइल नहीं।** इससे टर्मिनल सही डायरेक्टरी में शुरू होता है।

## 5. 💻 टर्मिनल को समझना

टर्मिनल एक टेक्स्ट-आधारित विंडो है जहां आप प्रोग्राम चलाने के लिए कमांड टाइप करते हैं।

### VS Code में टर्मिनल खोलना

- `` Ctrl + ` `` दबाएं (बैकटिक की, आमतौर पर `Esc` के नीचे)
- या मेनू बार से **Terminal → New Terminal** पर जाएं

### टर्मिनल के प्रकार

| टर्मिनल | आइकन/लेबल | कब उपयोग करें |
|----------|-----------|---------------|
| **PowerShell** | `PS C:\..>` | Windows पर डिफ़ॉल्ट, ठीक काम करता है |
| **Command Prompt** | `C:\..>` | यह भी काम करता है |

### ज़रूरी टर्मिनल कमांड

| कमांड | क्या करता है | उदाहरण |
|---------|-------------|---------|
| `cd folder_name` | एक फ़ोल्डर के अंदर जाएं | `cd Desktop` |
| `cd ..` | एक फ़ोल्डर पीछे जाएं | `cd ..` |
| `dir` | वर्तमान फ़ोल्डर की फ़ाइलें दिखाएं | `dir` |
| `cls` | स्क्रीन साफ़ करें | `cls` |
| `python file.py` | Python स्क्रिप्ट चलाएं | `python v2.py` |

> [!TIP]
> आप फ़ाइल नाम के **पहले कुछ अक्षर** टाइप करके **Tab** दबा सकते हैं — नाम अपने आप पूरा हो जाएगा।

## 6. 🚀 स्क्रिप्ट चलाना

### कदम दर कदम

1. **VS Code खोलें** प्रोजेक्ट फ़ोल्डर के साथ (खंड 4 देखें)
2. **टर्मिनल खोलें** (`` Ctrl + ` `` दबाएं)
3. **सत्यापित करें कि आप सही फ़ोल्डर में हैं।** टर्मिनल में वह पथ दिखना चाहिए जहां `v2.py` है। टाइप करें:
   ```powershell

   ```
   आपको सूची में `v2.py` दिखना चाहिए।

4. **स्क्रिप्ट चलाएं:**
   ```powershell
   python v2.py
   ```

5. **स्क्रीन पर दिए गए निर्देशों का पालन करें** — जब कहा जाए तब **Enter** दबाएं।

> [!CAUTION]
> स्क्रिप्ट चलते समय **टर्मिनल बंद न करें।** वर्चुअल एनवायरनमेंट बनाने और पैकेज इंस्टॉल करने में समय लगता है।

## 8. ✍️ सेटअप के बाद — अपना कोड लिखना

### कौन सी फ़ाइलें बनाई गईं?

| फ़ाइल | उद्देश्य | क्या आप संपादित कर सकते हैं? |
|------|---------|------------------------------|
| `model.py` | आपका सॉल्यूशन यहां लिखें | ✅ हां — यहां कोड लिखें |
| `data.py` | डेटासेट और ग्रेडिंग लॉजिक | ❌ नहीं — इसे छुएं नहीं! |

### `model.py` खोलना

1. VS Code में, **Explorer पैनल** (बाईं साइडबार) में `model.py` पर क्लिक करें
2. या `Ctrl + P` दबाएं और `model.py` टाइप करें

### `model.py` के अंदर क्या है?

आपने जो राउंड चुना उसके अनुसार:

**राउंड 1.1 (डेटा क्लीनिंग):**
```python
import data
RAW = data.RAW        # गंदा डेटासेट
cleaned = []          # साफ किया हुआ डेटा यहां डालें

# आपका कोड यहां — डेटा साफ करें!

data.graph(cleaned)   # यह लाइन हटाएं नहीं — यह आपके काम की जांच करती है
```

**राउंड 1.2 (sklearn लीनियर रिग्रेशन):**
```python
import data
import numpy as np
from sklearn.linear_model import LinearRegression
m, c = None, None     # ढलान (m) और अवरोधन (c) यहां रखें

# आपका कोड यहां — मॉडल ट्रेन करें!

data.graph(m, c)      # यह लाइन हटाएं नहीं — यह आपके काम की जांच करती है
```

**राउंड 2 (मैनुअल लीनियर रिग्रेशन):**
```python
import data
m, c = None, None
X_VALUES = data.X_VALUES
Y_VALUES = data.Y_VALUES

# आपका कोड यहां — sklearn के बिना m और c की गणना करें!

data.graph(m, c)      # यह लाइन हटाएं नहीं — यह आपके काम की जांच करती है
```

> [!WARNING]
> **नीचे की `data.graph(...)` लाइन कभी न हटाएं!** यह आपके सॉल्यूशन की जांच और ग्रेडिंग करती है।

## 9. ▶️ अपना सॉल्यूशन चलाना

`model.py` में कोड लिखने के बाद:

### तरीका 1: टर्मिनल से
```powershell
python model.py
```

### तरीका 2: टेस्ट कमांड का उपयोग (सुझाया गया)
```powershell
python v2.py --test
```
यह स्वचालित रूप से 3 काम करता है:
1. ✅ जांचता है कि `data.py` में छेड़छाड़ तो नहीं हुई
2. 🔄 अगर `data.py` बदली गई है तो उसे ठीक करता है
3. ▶️ आपकी `model.py` चलाता है

### अन्य उपयोगी फ्लैग

| कमांड | क्या करता है |
|---------|-------------|
| `python v2.py --guard` | जांचें कि `data.py` सही है या नहीं |
| `python v2.py --restore` | छेड़छाड़ हुई हो तो `data.py` को ठीक करें |
| `python v2.py --test` | Guard + Restore + model.py चलाएं |

### आउटपुट समझना

जब आप कोड चलाते हैं, तो एक **रिजल्ट पैनल** दिखेगा:

- ✔ **PERFECT** — आपका सॉल्यूशन सही है! 🎉
- ≈ **ACCEPTABLE** — करीब है, लेकिन सुधार चाहिए
- ✘ **INCORRECT** — अपने लॉजिक की जांच करें
- 🚨 **CHEAT DETECTED** — आपने राउंड 2 में प्रतिबंधित लाइब्रेरी का उपयोग किया

एक **matplotlib ग्राफ** भी खुलेगा जो आपके रिजल्ट विज़ुअली दिखाएगा।

**समस्या:** Python आपके सिस्टम PATH में नहीं है।

**समाधान:**
1. [python.org](https://www.python.org/downloads/) से Python फिर से इंस्टॉल करें
2. इंस्टॉलेशन के दौरान **✅ "Add Python to PATH"** पर टिक करें
3. इंस्टॉल करने के बाद VS Code रीस्टार्ट करें

**विकल्प:** `python` की जगह `python3` आज़माएं:
```powershell
python3 v2.py
```

---

### ❌ `ModuleNotFoundError: No module named 'venv'`

**समाधान:** पूर्ण Python वितरण इंस्टॉल करें (न्यूनतम नहीं)। python.org से फिर से डाउनलोड करें।

---

### ❌ टर्मिनल गलत डायरेक्टरी दिखा रहा है

**समाधान:** सुनिश्चित करें कि आपने VS Code में **फ़ोल्डर** खोला है, सिर्फ फ़ाइल नहीं। **File → Open Folder** पर जाएं और `v2.py` वाला फ़ोल्डर चुनें।

---

### ❌ `PermissionError` या `Access Denied`

**समाधान:** VS Code को Administrator के रूप में चलाएं:
1. Start Menu में VS Code पर राइट-क्लिक करें
2. "Run as administrator" चुनें

---

### ❌ अजीब अक्षर दिख रहे हैं (जैसे हरे रंग की जगह `[92m`)

**समस्या:** आपका टर्मिनल ANSI रंगों का समर्थन नहीं करता।

**समाधान:** पुराने Command Prompt की जगह **VS Code का इंटीग्रेटेड टर्मिनल** उपयोग करें। VS Code टर्मिनल रंगों को सही ढंग से संभालता है।

---

### ❌ `data.py has been modified or corrupted`

**समाधान:** रिस्टोर कमांड चलाएं:
```powershell
python v2.py --restore
```

---

### ❌ ग्राफ विंडो नहीं दिख रही

**समाधान:** सुनिश्चित करें कि `matplotlib` इंस्टॉल है। चलाएं:
```powershell
pip install matplotlib
```
या सेटअप स्क्रिप्ट फिर से चलाएं: `python v2.py`

### एंटी-चीट सिस्टम

स्क्रिप्ट में बिल्ट-इन एंटी-चीट व्यवस्था है:
- **data.py** में आपके कंप्यूटर के MAC एड्रेस + होस्टनेम से बनाया गया एक अद्वितीय सीड होता है
- इसका मतलब है कि हर विद्यार्थी को **अलग डेटासेट** मिलता है — आप दूसरों के जवाब कॉपी नहीं कर सकते
- **राउंड 2** में, `data.py` गुप्त रूप से आपकी `model.py` में प्रतिबंधित शब्दों को स्कैन करती है जैसे `sklearn`, `polyfit`, `scipy`, आदि
- अगर पाया गया → आपके ग्रेड की जगह 🚨 **CHEAT DETECTED** दिखाया जाएगा

### छुपी हुई सुविधा: सेटअप स्किप करें

अगर आपने पहले ही सेटअप चला लिया है और बस राउंड बदलना चाहते हैं, तो किसी भी "Press ENTER" प्रॉम्प्ट पर **`s`** (छोटा अक्षर) टाइप करें — सीधे राउंड चयन पर पहुंच जाएंगे।
