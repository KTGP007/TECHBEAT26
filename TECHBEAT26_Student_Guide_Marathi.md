## 2. ✅ सुरू करण्यापूर्वी काय लागते

सुरू करण्यापूर्वी, तुमच्याकडे हे असल्याची खात्री करा:

| आवश्यकता | कसे तपासावे |
|-----------|-------------|
| **Python 3.8+** इंस्टॉल | टर्मिनल उघडा → `python --version` टाइप करा |
| **VS Code** इंस्टॉल | Start Menu मध्ये "Visual Studio Code" शोधा |
| **स्क्रिप्ट फाइल** `v2.py` | आयोजकांकडून मिळेल |

### Python इंस्टॉल करणे (नसल्यास)

1. [python.org/downloads](https://www.python.org/downloads/) वर जा
2. Windows साठी नवीनतम Python 3 डाउनलोड करा
3. **⚠️ महत्त्वाचे:** इंस्टॉलेशन दरम्यान ✅ **"Add Python to PATH"** वर टिक करा
4. "Install Now" वर क्लिक करा
5. पडताळणी करा — Command Prompt उघडा आणि टाइप करा:
   ```
   python --version
   ```
   तुम्हाला असे काहीसे दिसेल: `Python 3.12.x`

### VS Code इंस्टॉल करणे (नसल्यास)

1. [code.visualstudio.com](https://code.visualstudio.com/) वर जा
2. Windows इंस्टॉलर डाउनलोड करा
3. डीफॉल्ट सेटिंग्जसह इंस्टॉल करा
4. **सूचना:** **Python extension** इंस्टॉल करा (पुढचा भाग पहा)

## 3. 🛠️ VS Code सेटअप

### टप्पा 1: Python Extension इंस्टॉल करा

1. VS Code उघडा
2. डाव्या साइडबारमध्ये **Extensions** आयकॉनवर क्लिक करा (4 चौरसांसारखे दिसते)
   - किंवा `Ctrl + Shift + X` दाबा
3. **"Python"** शोधा
4. **Microsoft** ने बनवलेल्यावर **Install** वर क्लिक करा (पहिला रिझल्ट असेल)

### टप्पा 2: Python Interpreter निवडा

1. `Ctrl + Shift + P` दाबा (Command Palette उघडेल)
2. **"Python: Select Interpreter"** टाइप करा
3. तुमचे इंस्टॉल केलेले Python व्हर्शन निवडा (उदा. `Python 3.12.x`)

## 4. 📂 प्रोजेक्ट फोल्डर उघडणे

### पद्धत 1: VS Code मधून

1. VS Code उघडा
2. **File → Open Folder** वर जा

1. File Explorer मध्ये `v2.py` असलेल्या फोल्डरमध्ये जा

```powershell
cd C:\path\to\your\folder
code .
```

## 5. 💻 टर्मिनल समजून घेणे

टर्मिनल ही एक टेक्स्ट-आधारित विंडो आहे जिथे तुम्ही प्रोग्राम चालवण्यासाठी कमांड टाइप करता.

### VS Code मध्ये टर्मिनल उघडणे

- `` Ctrl + ` `` दाबा (बॅकटिक की, सहसा `Esc` खाली असते)
- किंवा मेनू बारमधून **Terminal → New Terminal** वर जा

### टर्मिनलचे प्रकार

| टर्मिनल | आयकॉन/लेबल | कधी वापरावे |
|----------|-----------|-------------|
| **PowerShell** | `PS C:\..>` | Windows वर डीफॉल्ट, व्यवस्थित चालते |
| **Command Prompt** | `C:\..>` | हे देखील चालते |

### आवश्यक टर्मिनल कमांड

| कमांड | काय करते | उदाहरण |
|---------|----------|---------|
| `cd folder_name` | एका फोल्डरमध्ये जा | `cd Desktop` |
| `cd ..` | एक फोल्डर मागे जा | `cd ..` |
| `dir` | सध्याच्या फोल्डरमधल्या फाइल्स दाखवा | `dir` |
| `cls` | स्क्रीन साफ करा | `cls` |
| `python file.py` | Python स्क्रिप्ट चालवा | `python v2.py` |

> [!TIP]
> तुम्ही फाइलच्या नावाचे **पहिले काही अक्षरे** टाइप करून **Tab** दाबू शकता — नाव आपोआप पूर्ण होईल.

## 8. ✍️ सेटअप नंतर — तुमचा कोड लिहा

### कोणत्या फाइल्स तयार झाल्या?

| फाइल | उद्देश | तुम्ही बदल करू शकता का? |
|------|--------|------------------------|
| `model.py` | तुमचे सोल्यूशन इथे लिहा | ✅ हो — इथे कोड लिहा |
| `data.py` | डेटासेट आणि ग्रेडिंग लॉजिक | ❌ नाही — यालाहात लावू नका! |

### `model.py` उघडणे

1. VS Code मध्ये, **Explorer panel** (डावी साइडबार) मध्ये `model.py` वर क्लिक करा
2. किंवा `Ctrl + P` दाबा आणि `model.py` टाइप करा

### `model.py` मध्ये काय आहे?

तुम्ही निवडलेल्या राउंडनुसार:

**राउंड 1.1 (डेटा क्लीनिंग):**
```python
import data
RAW = data.RAW        # गोंधळलेला डेटासेट
cleaned = []          # स्वच्छ केलेला डेटा इथे ठेवा

# तुमचा कोड इथे — डेटा स्वच्छ करा!

data.graph(cleaned)   # ही ओळ काढू नका — ही तुमच्या कामाची तपासणी करते
```

**राउंड 1.2 (sklearn लिनिअर रिग्रेशन):**
```python
import data
import numpy as np
from sklearn.linear_model import LinearRegression
m, c = None, None     # उतार (m) आणि छेदनबिंदू (c) इथे ठेवा

# तुमचा कोड इथे — मॉडेल ट्रेन करा!

data.graph(m, c)      # ही ओळ काढू नका — ही तुमच्या कामाची तपासणी करते
```

**राउंड 2 (मॅन्युअल लिनिअर रिग्रेशन):**
```python
import data
m, c = None, None
X_VALUES = data.X_VALUES
Y_VALUES = data.Y_VALUES

# तुमचा कोड इथे — sklearn शिवाय m आणि c ची गणना करा!

data.graph(m, c)      # ही ओळ काढू नका — ही तुमच्या कामाची तपासणी करते
```

> [!WARNING]
> **खालची `data.graph(...)` ओळ कधीही काढू नका!** ही तुमच्या सोल्यूशनची तपासणी आणि ग्रेडिंग करते.

## 9. ▶️ तुमचे सोल्यूशन चालवणे

`model.py` मध्ये कोड लिहिल्यानंतर:

### पद्धत 1: टर्मिनलमधून
```powershell
python model.py
```

### पद्धत 2: टेस्ट कमांड वापरणे (सुचवलेले)
```powershell
python v2.py --test
```
हे आपोआप 3 गोष्टी करते:
1. ✅ `data.py` मध्ये छेडछाड झाली नाही ना ते तपासते
2. 🔄 `data.py` बदलली असल्यास ती दुरुस्त करते
3. ▶️ तुमची `model.py` चालवते

### इतर उपयुक्त फ्लॅग

| कमांड | काय करते |
|---------|----------|
| `python v2.py --guard` | `data.py` बरोबर आहे का ते तपासा |
| `python v2.py --restore` | छेडछाड झाल्यास `data.py` दुरुस्त करा |
| `python v2.py --test` | Guard + Restore + model.py चालवा |

### आउटपुट समजून घेणे

तुम्ही कोड चालवता तेव्हा, एक **रिझल्ट पॅनेल** दिसेल:

- ✔ **PERFECT** — तुमचे सोल्यूशन बरोबर आहे! 🎉
- ≈ **ACCEPTABLE** — जवळ आहे, पण सुधारणा हवी
- ✘ **INCORRECT** — तुमचा लॉजिक तपासा
- 🚨 **CHEAT DETECTED** — तुम्ही राउंड 2 मध्ये प्रतिबंधित लायब्ररी वापरली

एक **matplotlib ग्राफ** देखील उघडेल जो तुमचे रिझल्ट व्हिज्युअली दाखवेल.

## 11. 🔧 समस्या निवारण

### ❌ `'python' is not recognized as an internal or external command`

**समस्या:** Python तुमच्या सिस्टम PATH मध्ये नाही.

**उपाय:**
1. [python.org](https://www.python.org/downloads/) वरून Python पुन्हा इंस्टॉल करा
2. इंस्टॉलेशन दरम्यान **✅ "Add Python to PATH"** वर टिक करा
3. इंस्टॉल केल्यानंतर VS Code रीस्टार्ट करा

**पर्याय:** `python` ऐवजी `python3` वापरून बघा:
```powershell
python3 v2.py
```

---

### ❌ `ModuleNotFoundError: No module named 'venv'`

**उपाय:** पूर्ण Python वितरण इंस्टॉल करा (किमान नाही). python.org वरून पुन्हा डाउनलोड करा.

---

### ❌ टर्मिनल चुकीची डायरेक्टरी दाखवत आहे

**उपाय:** तुम्ही VS Code मध्ये **फोल्डर** उघडले आहे याची खात्री करा, फक्त फाइल नाही. **File → Open Folder** वर जा आणि `v2.py` असलेले फोल्डर निवडा.

---

### ❌ `PermissionError` किंवा `Access Denied`

**उपाय:** VS Code Administrator म्हणून चालवा:
1. Start Menu मध्ये VS Code वर राइट-क्लिक करा
2. "Run as administrator" निवडा

---

### ❌ विचित्र अक्षरे दिसत आहेत (जसे हिरव्या रंगाऐवजी `[92m]`)

**समस्या:** तुमचे टर्मिनल ANSI रंगांना सपोर्ट करत नाही.

**उपाय:** जुन्या Command Prompt ऐवजी **VS Code चे इंटिग्रेटेड टर्मिनल** वापरा. VS Code टर्मिनल रंग व्यवस्थित हाताळते.

---

### ❌ `data.py has been modified or corrupted`

**उपाय:** रिस्टोर कमांड चालवा:
```powershell
python v2.py --restore
```

---

### ❌ ग्राफ विंडो दिसत नाही

**उपाय:** `matplotlib` इंस्टॉल आहे याची खात्री करा. चालवा:
```powershell
pip install matplotlib
```
किंवा सेटअप स्क्रिप्ट पुन्हा चालवा: `python v2.py`
